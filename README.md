## How to use it


1. Run :

```
$ git clone https://github.com/Ademking/BulkSMSPython
$ cd BulkSMSPython
```

2. Modify numbers.txt with your numbers

3. Modify message.txt with your message body

4. Run :

```
$ app.py
```
